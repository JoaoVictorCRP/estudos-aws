# AppSync

- Serviço para conectar aplicativos a fontes de dados seguras através de **APIs GraphQL**.

- Permite que os desenvolvedores criem APIs flexíveis e eficientes, facilitando a obtenção de dados de várias fontes, como bancos de dados, serviços REST e outros.

- Também é muito utilizado para aplicações em tempo real, como chats e feeds de notícias, devido à sua capacidade de fornecer atualizações em tempo real aos clientes conectados.

- Um caso de uso comum deste serviço é em aplicativos móveis e web que exigem sincronização de dados em tempo real e acesso eficiente a dados distribuídos.
